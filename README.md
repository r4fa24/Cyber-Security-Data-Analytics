# Cyber Security Network Analytics 🛡️📊
A comprehensive analytical study of large-scale network traffic datasets to identify security patterns and performance metrics.

## 🚀 Project Overview
This project performs a deep-dive analysis of cybersecurity datasets to understand network behavior. By combining **Python** for automated data wrangling and **Excel** for advanced statistical modeling, I transformed raw network logs into actionable security insights.
![Cyber Security Dashboard](Dashboard_visuals
.png)

## ✨ Key Technical Insights
* **Exploratory Data Analysis (EDA):** Performed descriptive statistics on 1,779 network instances to analyze APP_BYTES and TCP conversation exchanges.
* **Statistical Modeling:** Calculated advanced metrics including **Kurtosis (1766.4)** and **Skewness (41.9)**, identifying a Leptokurtic distribution indicative of extreme network traffic outliers.
* **Geospatial Traffic Profiling:** Utilized Pivot Tables to map global traffic, identifying the US and Canada as primary throughput contributors.
* **Infrastructure Benchmarking:** Analyzed DNS query times and application bytes across various server architectures (Nginx, Apache, Cloudflare).

## 🛠️ Tech Stack
* **Analysis:** Python (Pandas), Excel (Pivot Tables, Dashboards)
* **Statistics:** Descriptive Analysis, Standard Deviation, Kurtosis, and Skewness modeling.
